# 축구 경기 결과 예측 알람 서비스

## 프로젝트 소개
축구 경기의 첫 배당과 경기 시작 5분전 배당을 비교하여 결과를 예측하는 서비스입니다.
<br>

### 🖥️ 개발 환경
- `Python 3.7`
- AWS Region : `ap-northeast-2`
